Java Introduction

What is Java?

Java is a high-level, class-based, object-oriented programming language that is designed to have as few implementation dependencies as possible.

  • Platform-independent
  • Object-oriented
  • Robust and secure

What can Java do?

  • Java can be used to create web applications.
  • Java can be used for mobile applications (Android).
  • Java can be used for enterprise-level applications.

Why Java?

  • Java is platform-independent due to the Java Virtual Machine (JVM).
  • Java has a rich API and a large community.
  • Java is widely used in enterprise environments.

Good to know

  • The latest version of Java is Java 17 (as of 2023).
  • Java can be written in any text editor or IDE like Eclipse, IntelliJ IDEA, etc.

Java Features

Java provides several features that make it a popular programming language. Here are some essential features:

1) Object-Oriented

Java is an object-oriented programming language, which means it is based on objects and classes.

2) Platform-Independent

Java code can run on any device that has the Java Virtual Machine (JVM) installed.

3) Robust

Java has strong memory management and exception handling features, making it robust.

4) Multithreaded

Java supports multithreading, allowing concurrent execution of two or more threads.

5) Secure

Java provides a secure environment for developing applications through its security features.

Java Variables

A variable in Java is a container that holds data that can be changed during the execution of a program.

Java is a statically typed language, meaning you must declare a variable's type before using it.

Types of Variables

  • Local Variables
  • Instance Variables
  • Static Variables

Declaring Variables

In Java, you declare a variable by specifying its type followed by its name:

int number; // Declaration number = 10; // Initialization

Java Operators

Operators in Java are used to perform operations on variables and values. Java supports various types of operators:

1) Arithmetic Operators

Used to perform basic mathematical operations like addition, subtraction, multiplication, and division.

int a = 10, b = 5; int sum = a + b; // Addition

2) Relational Operators

Used to compare two values. Examples include ==, !=, >, <, >=, and <=.

if (a > b) { System.out.println("a is greater than b"); }

3) Logical Operators

Used to combine multiple conditions. Examples include && (AND), || (OR), and ! (NOT).

if (a > 0 && b > 0) { System.out.println("Both a and b are positive"); }

Java Control Statements

Control statements in Java are used to control the flow of execution of the program. They include:

1) If-Else Statement

Used to execute a block of code based on a condition.

if (a > b) { System.out.println("a is greater than b"); } else { System.out.println("b is greater than a"); }

2) Switch Statement

Used to execute one block of code among many based on the value of a variable.

switch (day) { case 1: System.out.println("Monday"); break; case 2: System.out.println("Tuesday"); break; default: System.out.println("Invalid day"); }

3) Loops

Used to execute a block of code repeatedly. Examples include for, while, and do-while loops.

for (int i = 0; i < 5; i++) { System.out.println(i); }

Java Strings

Strings in Java are objects that represent sequences of characters. They are immutable, meaning their values cannot be changed after creation.

Creating Strings

Strings can be created using string literals or the new keyword.

String str1 = "Hello"; // Using string literal String str2 = new String("World"); // Using new keyword

String Methods

Java provides various methods to manipulate strings, such as length(), charAt(), substring(), and equals().

String str = "Java Programming"; int len = str.length(); // Returns the length of the string

Java Arrays

Arrays in Java are used to store multiple values of the same type in a single variable. They are fixed in size once declared.

Declaring Arrays

Arrays can be declared using the following syntax:

int[] numbers = new int[5]; // Declares an array of size 5

Accessing Array Elements

Array elements can be accessed using their index, starting from 0.

numbers[0] = 10; // Assigns 10 to the first element int firstElement = numbers[0]; // Accesses the first element

Java Collections

The Java Collections Framework provides a set of classes and interfaces to store and manipulate groups of objects. It includes:

1) List

An ordered collection that allows duplicate elements. Examples include ArrayList and LinkedList.

List list = new ArrayList<>(); list.add("Java"); list.add("Python");

2) Set

A collection that does not allow duplicate elements. Examples include HashSet and TreeSet.

Set set = new HashSet<>(); set.add("Java"); set.add("Python");

3) Map

A collection that stores key-value pairs. Examples include HashMap and TreeMap.

Map map = new HashMap<>(); map.put("Java", 1); map.put("Python", 2);

Java Exception Handling

Exception handling in Java is used to handle runtime errors and maintain the normal flow of the program. It involves:

1) Try-Catch Block

Used to catch and handle exceptions.

try { int result = 10 / 0; // Division by zero } catch (ArithmeticException e) { System.out.println("Cannot divide by zero"); }

2) Finally Block

Used to execute code regardless of whether an exception occurs or not.

try { // Code that may throw an exception } finally { System.out.println("This will always execute"); }

3) Throw and Throws

Used to explicitly throw an exception and declare exceptions that a method might throw.

void checkAge(int age) throws Exception { if (age < 18) { throw new Exception("Age must be 18 or above"); } }

Java Multithreading

Multithreading in Java allows concurrent execution of two or more threads. It is used to achieve multitasking.

1) Creating Threads

Threads can be created by extending the Thread class or implementing the Runnable interface.

class MyThread extends Thread { public void run() { System.out.println("Thread is running"); } }

2) Thread Synchronization

Used to control the access of multiple threads to shared resources.

synchronized void printNumbers() { for (int i = 1; i <= 5; i++) { System.out.println(i); } }

3) Thread Lifecycle

Threads go through various states: New, Runnable, Blocked, Waiting, Timed Waiting, and Terminated.
